FAQ

What's a Banding Code?

Banding Codes are short, four-letter codes used to abbreviate the names of bird species. Codes are used both by ornithologists and birders as a shorthand, mostly to make field notes much easier.

How are they formed?

The basic rules are as follows...

For 1-word species:

The first four letters are used. "Mallard" becomes "MALL".

For 2-word species:

The first two letters of each word are combined. "Blue Jay" becomes "BLJA".

For 3-word species:

The first letter of the first two words are combined with the first two letters of the last word. "Black-capped Chickadee" becomes "BCCH".

For 4-word species:

The first letter of each word is combined. "Black-crowned Night-Heron" becomes "BCNH".

If it's that easy, why this tool?

It's actually not that easy. Some species end up with the same code, using these rules. For example, "Canyon Wren", "Carolina Wren", and "Cactus Wren" all end up with the code "CAWR". Oops. So secondary rules are made.. and sometimes those rules don't work either, and tertiary rules are made. It's those exceptions that can make remembering banding codes tough.

I found a bird band, or photographed a banded bird! What do I do?

As long as it wasn't an obviously domesticated bird, go report it to USGS/Environment Canada via this site. Also, you should report it (and all your bird sightings, really) on eBird.

Why did you make this tool?

There didn't appear to be any good sites or online tools that had this kind of quick lookup functionality!
